# SistemaPeaje

Este proyecto es un sistema de gestión de peajes desarrollado en Java.

## Estructura del Proyecto

- `src/com/uide/autonomo/tres/modelo/` contiene las clases principales del sistema:
  - `Camion.java`
  - `Carro.java`
  - `Moto.java`
  - `Peaje.java`
  - `SistemaPeaje.java`
  - `Vehiculo.java`

## Ejecución

Compila y ejecuta la clase `PruebaFiguras` para ver ejemplos de uso de las figuras geométricas:

```powershell
# Compilar
javac src/com/uide/autonomo/tres/modelo/*.java src/com/uide/autonomo/tres/SistemaPeaje.java

# Ejecutar
java -cp src com.uide.autonomo.tres.SistemaPeaje
```
